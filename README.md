# Fitness Bot

Телеграм-бот для фитнес-клуба с возможностями персонализации программ тренировок, ведения учёта посещений и отслеживания прогресса клиентов.

## Возможности
- Автоматическая регистрация клиента по команде `/start` с интерактивным онбордингом (цель, уровень подготовки, предпочтительные дни тренировок).
- Персональные рекомендации по тренировкам и питанию на основе выбранной цели и уровня.
- Учёт посещений и автоматическое уведомление клиента после отметки администратором.
- Отслеживание прогресса: клиент может фиксировать вес и заметки, последние записи доступны прямо в чате.
- Админ-панель в один клик: список клиентов, статистика, отметка посещений, добавление новых клиентов.
- Команда `/broadcast` для рассылки объявлений всем пользователям.

## Подготовка
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создайте бота у [@BotFather](https://t.me/BotFather) и замените значение `TOKEN` в `main.py` на токен вашего бота.
3. Укажите свой Telegram ID в переменной `ADMIN_ID` в `main.py`.

## Запуск
```bash
python main.py
```

Бот автоматически создаст файл `visits.db` в корне проекта и выполнит все необходимые миграции.

## Использование
- Клиенту достаточно отправить `/start`, пройти короткий онбординг и воспользоваться меню.
- Администратор получает доступ к расширенной клавиатуре и может управлять клиентами и их посещениями.
